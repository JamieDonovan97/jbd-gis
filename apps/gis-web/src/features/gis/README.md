# features/gis

A full-screen map with floating control surfaces.

| Path          | Holds                                                             |
| ------------- | ----------------------------------------------------------------- |
| `GisView.tsx` | The route component.                                              |
| `components/` | Map canvas, navigation, tool dock, inspector, and layer renderer. |
| `config/`     | Tool and layer registries.                                        |
| `store/`      | View state (Zustand).                                             |
