# components

Shared, view-agnostic building blocks.

| Path     | Holds                                                                    |
| -------- | ------------------------------------------------------------------------ |
| `ui/`    | Radix + cva primitives (button, badge, tooltip, scroll-area, separator). |
| `glass/` | The frosted-glass surface that floating panels compose.                  |

Add primitives with `npx shadcn add <name>`.
